### 第二步：完成 KYC 认证

注册完进 APP，点「会员」→「KYC」，虚拟卡只需要做 Level 1 认证，身份证 + 人脸识别就行，不需要护照、不需要海外银行流水。整个过程 3–5 分钟，提交完一般几分钟到几小时审核通过。

> 这里有个细节：PokePay 的 L1 认证严格来说不需要信息完全真实，但实测用真实身份证通过率更高、后续封号风险更小。如果你打算后续升级实体卡（L2 必须真实），从一开始就用真实信息最省事。

### 第三步：充值 USDT

点「钱包」→「充币」，选 USDT，网络建议选 **TRC20**（Tron），Gas 费最低、到账最快，一般 5–15 分钟到账。首次充值建议充 25–30 USDT，这样开卡 + 第一笔订阅手续费都够用。

### 第四步：申请虚拟卡

钱到账后，点「卡片」→「申请卡片」→「虚拟卡」，选你想开的卡段（Visa 或 Mastercard），开卡费会从账户余额里扣。用邀请券抵扣后，PokeCard Visa 虚拟卡实际扣 7.2 USDT。

这是新手最容易出问题的一步，单独拎出来说。

**提币网络怎么选**：PokePay 充值时选 TRC20，那么你在交易所提币时也要选 TRC20，两边网络必须一致。如果一边选 TRC20 一边选 ERC20，币可能就找不回来了。

A：不支持。PokePay 只接受 USDT、USDC、BTC、ETH 充值，人民币需要先在交易所买 USDT 再提币过来。

**Q：能在国内扫码消费吗？**
